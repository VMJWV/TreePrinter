实现参考 Milo Yip https://zhuanlan.zhihu.com/p/38013510

对着大佬的实现加了点代码魔改了一遍

让支持更为广泛 让其对其更加平整

支持C、C++实现的二叉链表实现的二叉树

可以把二叉树打印成一个比较好看的树状
